# Road Vision — система моніторингу стану дорожнього покриття

Навчальний проєкт з дисципліни «Розробка та експлуатація систем Інтернет речей».
IoT-система збору, обробки, зберігання та візуалізації даних про стан дорожнього
покриття. Дані збираються з датчиків автомобіля (акселерометр, GPS) і проходять
шлях **Agent → Edge → Hub → Store → MapView**.

## Архітектура

| Компонент | Призначення | Лабораторна |
|---|---|---|
| **agent** | збір даних з датчиків (CSV) і відправка на Edge через MQTT | ЛР1 |
| edge | первинна обробка даних (визначення стану дороги) | ЛР2+ |
| hub | накопичення та збереження даних у БД | ЛР3+ |
| store | API для зберігання/отримання даних | ЛР3+ |
| MapView | візуалізація результатів | ЛР4+ |

## Стек

Java 25 · Maven · Eclipse Paho (MQTT) · Jackson · Docker / Docker Compose
